## Obtener Servidores por Tipo

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/servers/getByType
```

### Parámetros de Query

| Key  | Value                                  | Description                         |
| ---- | -------------------------------------- | ----------------------------------- |
| `id` | `b5ad9754-0419-46e5-b556-7ab1ab8db3e1` | ID del tipo de servidor a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
