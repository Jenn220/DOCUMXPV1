## Obtener Servidor por Código

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/servers/getByCode
```

### Parámetros de Query

| Key    | Value  | Description                     |
| ------ | ------ | ------------------------------- |
| `code` | `S001` | Código del servidor a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
