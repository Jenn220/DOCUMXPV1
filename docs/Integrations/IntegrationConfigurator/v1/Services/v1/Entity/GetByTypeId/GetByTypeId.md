## Obtener Entidades por ID de Repositorio

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/entities/getByRepositoryId
```

### Parámetros de Query

| Key            | Value                                  | Description                                     |
| -------------- | -------------------------------------- | ----------------------------------------------- |
| `repositoryId` | `059bb531-5b34-4435-bdad-21c81b41cac1` | ID del repositorio para consultar sus entidades |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
