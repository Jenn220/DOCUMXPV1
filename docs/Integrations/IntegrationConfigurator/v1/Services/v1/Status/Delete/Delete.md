## Eliminar Estado

### Método HTTP

`DELETE`

### URL

```
{{server_orchestrator}}/api/v1/status/delete
```

### Parámetros de Query

| Key  | Value                                  | Description                    |
| ---- | -------------------------------------- | ------------------------------ |
| `id` | `3fa85f64-5717-4562-b3fc-2c963f66afa6` | ID único del estado a eliminar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
