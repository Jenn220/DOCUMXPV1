## Obtener Adaptador por Código

### Método HTTP

`GET`

### URL

```
{{server_orchestrador}}/api/v1/adapters/getByCode
```

### Parámetros de Query

| Key    | Value  | Description                            |
| ------ | ------ | -------------------------------------- |
| `code` | `E001` | Código único del adaptador a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
