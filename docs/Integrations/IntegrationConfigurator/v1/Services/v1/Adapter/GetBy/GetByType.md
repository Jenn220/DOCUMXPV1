## Obtener Adaptador por Tipo

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/adapters/getByCode
```

### Parámetros de Query

| Key    | Value     | Description                              |
| ------ | --------- | ---------------------------------------- |
| `code` | `EXTRACT` | Código de tipo del adaptador a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
