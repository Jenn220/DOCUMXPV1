## Eliminar Repositorio

### Método HTTP

`DELETE`

### URL

```
{{server_orchestrator}}/api/v1/repositories/delete
```

### Parámetros de Query

| Key  | Value                                  | Description                         |
| ---- | -------------------------------------- | ----------------------------------- |
| `id` | `6bd36e91-c0de-4b6d-87b8-5dea7f2c2274` | ID único del repositorio a eliminar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
