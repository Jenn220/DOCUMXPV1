## Obtener Repositorio por Código

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/repositories/getByCode
```

### Parámetros de Query

| Key    | Value  | Description                        |
| ------ | ------ | ---------------------------------- |
| `code` | `R001` | Código del repositorio a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
