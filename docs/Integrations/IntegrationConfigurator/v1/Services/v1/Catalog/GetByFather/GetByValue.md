## Obtener Catálogo por Valor

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/catalogs/GetByValue
```

### Parámetros de Query

| Key     | Value | Description                    |
| ------- | ----- | ------------------------------ |
| `value` | ``    | Valor del catálogo a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
