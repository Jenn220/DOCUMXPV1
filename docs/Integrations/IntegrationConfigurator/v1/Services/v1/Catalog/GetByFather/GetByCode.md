## Obtener Catálogo por Código

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/catalogs/GetByCode
```

### Parámetros de Query

| Key    | Value | Description                     |
| ------ | ----- | ------------------------------- |
| `code` | `1`   | Código del catálogo a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
