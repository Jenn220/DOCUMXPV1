## Eliminar Catálogo

### Método HTTP

`DELETE`

### URL

```
{{server_orchestrator}}/api/v1/catalogs/Delete
```

### Parámetros de Query

| Key  | Value                                  | Description                      |
| ---- | -------------------------------------- | -------------------------------- |
| `id` | `51faf974-5c9e-4936-b234-91a173356dea` | ID único del catálogo a eliminar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
