### URL

```
{{server_orchestrator}}/api/v1/synchronizationStates/delete
```

| ---- | -------------------------------------- | ------------------------------------------ |
| `id` | `b5ad9754-0419-46e5-b556-7ab1ab8db3e1` | ID del estado de sincronización a eliminar |
