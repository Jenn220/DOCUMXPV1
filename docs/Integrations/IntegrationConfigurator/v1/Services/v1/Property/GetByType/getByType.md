## Obtener Propiedades por Tipo

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/properties/getByType
```

### Parámetros de Query

| Key  | Value                                  | Description                                |
| ---- | -------------------------------------- | ------------------------------------------ |
| `id` | `059bb531-5b34-4435-bdad-21c81b41cac1` | ID del tipo para consultar sus propiedades |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
