## Eliminar Propiedad

### Método HTTP

`DELETE`

### URL

```
{{server_orchestrator}}/api/v1/properties/delete
```

### Parámetros de Query

| Key  | Value                                  | Description                         |
| ---- | -------------------------------------- | ----------------------------------- |
| `id` | `059bb531-5b34-4435-bdad-21c81b41cac1` | ID único de la propiedad a eliminar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
