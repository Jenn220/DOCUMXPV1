## Obtener Conexión por Tipo

### Método HTTP

`GET`

### URL

```
{{server_orchestrator}}/api/v1/connections/GetByType
```

### Parámetros de Query

| Key    | Value | Description                  |
| ------ | ----- | ---------------------------- |
| `type` | ``    | Tipo de conexión a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
