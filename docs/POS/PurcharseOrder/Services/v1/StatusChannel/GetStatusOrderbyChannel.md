## Obtener Estado de Orden por Canal

### Método HTTP

`GET`

### URL

```
{{server_purchaseOrders}}/api/v1/statuschannel/get_status_order_by_channel
```

### Parámetros de Query

| Key     | Value    | Descripción                  |
| ------- | -------- | ---------------------------- |
| channel | Delivery | Nombre del canal a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
