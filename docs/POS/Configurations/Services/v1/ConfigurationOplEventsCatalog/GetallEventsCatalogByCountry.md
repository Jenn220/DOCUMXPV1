## Obtener Todos los Eventos Configurados por País

### Método HTTP

`GET`

### URL

```
{{server_POS_Configurations}}/api/v1/configurationopleventscatalog/get_all_configured_events_by_country_id
```

### Parámetros de Query

| Key        | Value                                | Descripción                  |
| ---------- | ------------------------------------ | ---------------------------- |
| country_id | d6cac83d-6ce5-f62e-2628-1e2e9ae7b51f | Identificador único del país |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
