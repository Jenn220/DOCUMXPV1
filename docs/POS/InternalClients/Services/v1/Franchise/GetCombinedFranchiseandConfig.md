## Obtener Franquicia Combinada con Configuración

### URL

```
{{server_internalClients}}/api/v1/franchise/get_combined_franchise_and_config
```

### Parámetros de Query

| Key       | Value                                | Descripción                    |
| --------- | ------------------------------------ | ------------------------------ |
| franchise | bc2d8ada-e25e-1bb7-55fe-32d1205ac4af | Identificador de la franquicia |
