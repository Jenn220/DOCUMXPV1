## Agregar Autorización de Notificación

### Método HTTP

`GET`

### URL

```
{{server_pwa}}/api/v1/add_notification_authorization
```

### Parámetros de Query

| Key         | Value                | Descripción                |
| ----------- | -------------------- | -------------------------- |
| uuidconnect | channel-connect-uuid | UUID de conexión del canal |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
