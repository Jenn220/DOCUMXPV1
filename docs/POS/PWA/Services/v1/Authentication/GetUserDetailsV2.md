## Obtener Detalles de Usuario v2

### Método HTTP

`GET`

### URL

```
{{server_pwa}}/hera/api/v1/Authentication/get-user-details-v2
```

### Parámetros de Query

| Key     | Value   | Descripción                    |
| ------- | ------- | ------------------------------ |
| account | USER123 | Cuenta del usuario a consultar |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
