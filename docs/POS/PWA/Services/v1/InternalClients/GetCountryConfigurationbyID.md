## Obtener Configuración de País por ID

### Método HTTP

`GET`

### URL

```
{{server_pwa}}/internalClients/api/v1/country/get_country_configuration_by_id
```

### Headers

| Key          | Value            | Descripción                       |
| ------------ | ---------------- | --------------------------------- |
| Content-Type | application/json | Tipo de contenido de la solicitud |
| is_download  | is_download      | Indicador de descarga             |

### Parámetros de Query

| Key     | Value                                | Descripción            |
| ------- | ------------------------------------ | ---------------------- |
| country | d6cac83d-6ce5-f62e-2628-1e2e9ae7b51f | Identificador del país |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
