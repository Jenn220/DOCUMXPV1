## Obtener Eventos Configurados por País

### Método HTTP

`GET`

### URL

```
{{server_pwa}}/YOUR_REDIS_OR_AZURE_KEY_HEREget_all_configured_events_by_country_id
```

### Parámetros de Query

| Key        | Value                                | Descripción            |
| ---------- | ------------------------------------ | ---------------------- |
| country_id | d6cac83d-6ce5-f62e-2628-1e2e9ae7b51f | Identificador del país |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
