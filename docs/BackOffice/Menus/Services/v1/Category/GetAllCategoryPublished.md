## Obtener Todas las Categorías Publicadas

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/api/v1/category/getallcategorypublished
```

### Parámetros de Query

| Key         | Value                                  | Description         |
| ----------- | -------------------------------------- | ------------------- |
| `franchise` | `bc2d8ada-e25e-1bb7-55fe-32d1205ac4af` | ID de la franquicia |
| `country`   | `d6cac83d-6ce5-f62e-2628-1e2e9ae7b51f` | ID del país         |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
