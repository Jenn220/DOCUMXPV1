## Obtener Pregunta Sugerida por ID

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/api/v1/suggestedquestion/getbyid
```

### Parámetros de Query

| Key                     | Value                                  | Description                |
| ----------------------- | -------------------------------------- | -------------------------- |
| `suggested_question_id` | `3fa85f64-5717-4562-b3fc-2c963f66afa6` | ID de la pregunta sugerida |
| `franchise`             | `bc2d8ada-e25e-1bb7-55fe-32d1205ac4af` | ID de la franquicia        |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
