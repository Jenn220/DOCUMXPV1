## Obtener Información Básica del Menú

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/api/v1/menu/getbasicinfo
```

### Parámetros de Query

| Key       | Value                                  | Description |
| --------- | -------------------------------------- | ----------- |
| `menu_id` | `902499bf-284f-4f2e-80b7-60ef60c6bd9c` | ID del menú |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
