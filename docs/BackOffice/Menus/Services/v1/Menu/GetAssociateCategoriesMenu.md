## Obtener Categorías Asociadas al Menú

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/api/v1/menu/get_asocciate_categories_menu
```

### Parámetros de Query

| Key         | Value                                  | Description         |
| ----------- | -------------------------------------- | ------------------- |
| `menu_id`   | `902499bf-284f-4f2e-80b7-60ef60c6bd9c` | ID del menú         |
| `franchise` | `bc2d8ada-e25e-1bb7-55fe-32d1205ac4af` | ID de la franquicia |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
