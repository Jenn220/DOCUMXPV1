## Obtener Producto con Detalles de Preguntas Sugeridas

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/YOUR_REDIS_OR_AZURE_KEY_HERE
```

### Parámetros de Query

| Key      | Value    | Description |
| -------- | -------- | ----------- |
| `plu_id` | `string` | ID del PLU  |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
