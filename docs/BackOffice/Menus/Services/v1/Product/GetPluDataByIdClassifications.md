## Obtener Datos de PLU por Clasificaciones

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/YOUR_REDIS_OR_AZURE_KEY_HERE
```

### Parámetros de Query

| Key                   | Value                                  | Description            |
| --------------------- | -------------------------------------- | ---------------------- |
| `classifications_ids` | `string`                               | IDs de clasificaciones |
| `id_franchise`        | `bc2d8ada-e25e-1bb7-55fe-32d1205ac4af` | ID de la franquicia    |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
