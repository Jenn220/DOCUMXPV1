## Obtener Producto por ID con Detalles de Categorías

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/YOUR_REDIS_OR_AZURE_KEY_HERE
```

### Parámetros de Query

| Key         | Value    | Description         |
| ----------- | -------- | ------------------- |
| `MenuPluId` | `string` | ID del PLU del menú |

### Cuerpo de la Solicitud

No requiere cuerpo en la solicitud.
