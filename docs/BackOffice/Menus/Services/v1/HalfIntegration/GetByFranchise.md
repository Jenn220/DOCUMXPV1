## Obtener Integración Parcial por Franquicia

### Método HTTP

`GET`

### URL

```
{{server_BO_Menus}}/api/v1/halfintegration/getbyfranchise
```

### Parámetros de Query

| Key            | Value                                  | Description         |
| -------------- | -------------------------------------- | ------------------- |
| `franchise_id` | `bc2d8ada-e25e-1bb7-55fe-32d1205ac4af` | ID de la franquicia |
